# User Guide - Spinning Wheel

## Language

[English](USER_GUIDE.md) | [Español](USER_GUIDE.es.md)

## Overview

Spinning Wheel lets you define challenge options, spin randomly, and optionally run in turn mode where used options are eliminated after each spin.

## Main Controls

- `Language`: switch EN/ES
- `Sound`: mute/unmute effects
- `Turn mode`: eliminate selected options after each result
- `Fullscreen`: toggle fullscreen window
- `Settings`: open/close side drawer

## Basic Flow

1. Click `Spin!` (or press `Enter`).
2. Wait for the wheel to stop.
3. Review the selected challenge in the result overlay.
4. Click `Continue` (or press `Enter`) to keep playing.

## Settings Drawer

Inside Settings you can:

- Choose a color palette.
- Add a challenge (max 30 chars).
- Edit challenge text directly.
- Remove a challenge.
- `Reset`: restore default challenge list.
- `Restart`: clear eliminations and restart current set.

## Turn Mode

When turn mode is active:

1. Spin the wheel.
2. The selected option is marked for elimination after `Continue`.
3. Repeat until no options remain.
4. Final completion modal appears.

## Keyboard Shortcuts

- `Enter`
- In input field: add option.
- Outside input: spin / continue / play again.
- `Esc`
- Close settings drawer.
- Close result overlay.

## Troubleshooting

- No sound: interact once with the app (click/keyboard), check mute icon and OS volume.
- Wheel does not spin: ensure at least one active option and no blocking modal is open.
- Changes not persisted: verify localStorage is enabled for the app context.
